# WebRTC 局域网 P2P 通讯工具

一款专为局域网环境设计的现代化、零依赖的 WebRTC 端对端通讯工具。

## 📖 项目简介

本项目基于原生 HTML5、CSS3 和 JavaScript 构建，无需安装任何客户端，打开浏览器即可使用。通过 WebRTC 技术，实现局域网内的设备进行安全的 P2P（点对点）数据传输和音视频通话。

由于采用 P2P 架构，数据不经过第三方服务器，传输速度极快，隐私安全性高，非常适合家庭内网、公司局域网或无互联网环境下的设备互联需求。

## ✨ 核心功能

### 1. 💬 即时通讯 (IM)
- **富文本编辑**：支持文字、Emoji 表情混合输入。
- **图片传输**：支持粘贴剪贴板图片、拖拽发送，内置图片预览器（支持缩放、拖拽移动）。
- **消息分片**：自动处理超大文本消息的分片发送与重组。

### 2. 📁 高速文件传输
- **大文件支持**：基于 DataChannel 的大文件分片传输技术，轻松发送 GB 级文件。
- **二进制协议**：采用高效的二进制协议封装（非 JSON Base64），减少 50%+ 的数据冗余，显著提升传输效率。
- **流控机制**：智能缓冲区管理，防止内存溢出，保障传输流畅。
- **实时进度**：可视化传输进度条，实时显示速度与百分比。

### 3. 📹 音视频通话
- **多源支持**：前置/后置摄像头切换、屏幕共享。
- **高清画质**：支持自动分辨率适配，最高可达 4K/8K 画质。
- **灵活窗口**：视频窗口支持拖拽、缩放、全屏模式。
- **码率控制**：可手动调节视频码率，适应不同网络环境。

### 4. 🔌 灵活的连接方式
- **服务器模式**：支持自建轻量级信令服务器，适合局域网多设备使用。
- **手动模式**：无需服务器！支持手动复制粘贴 SDP/ICE 信息进行连接，适配物理隔离网络。

## 🛠️ 技术架构

- **前端框架**：原生 JavaScript (Vanilla JS)，无第三方依赖库。
- **通信协议**：WebRTC (RTCPeerConnection, DataChannel, getUserMedia)。
- **信令机制**：
  - 自动模式：HTTP 轮询 / LocalStorage 模拟
  - 手动模式：人工复制粘贴 (离线可用)
- **文件传输**：自定义二进制分片协议 + ArrayBuffer。

## 🚀 快速开始

### 方式一：直接打开（同一台电脑测试）
1. 双击打开 `index.html` 文件。
2. 打开浏览器控制台 (F12) 或直接在页面操作。
3. 打开两个标签页，分别选择角色为“发起者”和“接收者”，点击连接即可通讯。
